# Run in Your Book

## Install:

Please install [LanFilePackage](https://langong-dev.github.io/LanFilePackage/) first.

Then input this:

```bash
./LanGong install github langong-dev/Zff-Online
```

## Run:

```bash
./LanGong run Zff-Online
```
